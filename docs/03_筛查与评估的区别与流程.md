# 筛查与评估:区别与流程

家长经常混淆"筛查"和"评估",但这两者在目的、方法和结论深度上有显著差异。

## 筛查(Screening)

**目的**:快速识别"可能存在风险"的儿童,作为进一步评估的前置环节。

**特点**:
- 时间短(通常 10-30 分钟)
- 可以通过游戏化、自动化工具进行
- 结果是**风险提示**,不是诊断
- 适合大范围使用,成本低

**输出**:通常给出"低风险 / 中等风险 / 高风险"之类的分层结果,而非具体诊断。

**典型筛查工具涉及的维度**:
- 语音意识任务(如押韵判断、音节删除)
- 快速命名(RAN,Rapid Automatized Naming)
- 字词识别任务
- 简短阅读流畅性测试

## 评估(Assessment)

**目的**:在筛查出风险后,由专业人员对儿童进行全面的、标准化的能力诊断。

**特点**:
- 时间长(通常 2-4 小时,可能分多次)
- 由具备资质的专业人员进行
- 结果是**诊断性结论**,包含具体能力画像
- 输出完整的评估报告

**评估维度通常包括**:
- 智力水平(排除智力因素)
- 阅读流畅性、准确性、理解力
- 书写能力
- 拼写/听写
- 音素意识、字形加工
- 记忆、注意力等认知功能
- 情绪与行为状态

## 从筛查到干预的完整流程

```
家长观察到困难
      ↓
初步筛查(游戏化工具/问卷)
      ↓
[有风险] → 专业评估
      ↓
评估报告 + 能力画像
      ↓
制定个性化干预方案
      ↓
干预训练(3-12 个月或更长)
      ↓
阶段性复评 + 方案调整
```

## 家长常见问题

**Q: 筛查结果说"高风险",是不是就确诊读写障碍了?**
A: 不是。筛查只是提示需要进一步评估。约 70%-80% 的高风险筛查结果会在评估中被确认,但仍有一部分会被排除(例如发现是注意力问题或其他原因)。

**Q: 评估会不会给孩子贴标签?**
A: 专业评估报告的目的是**理解孩子的认知特点**,给出针对性支持方案,不是贴标签。国内大部分专业机构的评估报告只会给家长和教学支持人员查阅,不会进入学校档案。

**Q: 评估之后一定要做干预吗?**
A: 取决于评估结果。如果确认为读写障碍,越早干预效果越好。如果是其他原因(如教学缺失、注意力问题),也会有针对性建议。

**Q: 筛查和评估的结果有时效性吗?**
A: 儿童发展是动态的,通常建议 6-12 个月复评一次,观察干预效果和能力变化。
